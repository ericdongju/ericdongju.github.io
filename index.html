WHO AM I?

DESIGNER

I enjoy simple and ergonomic designs. It doesn't matter how technically impressive the UI may be, if it doesn't resonate with the users in a personal and comfortable level, the design is flawed. Instead, I strive for an unique balance between technicality and usability.

CREATOR

I started writing stories when I was five. I began coding when I thirteen. Creation is something I've always been passionate about. I don't think having an arsenal of coding languages makes you a great programmer. I believe it's the willingness to create and inspire others that makes you great.

DEVELOPER

I specialize in back-end and front-end development, along with programming languages such as Python and Racket. When it comes to development, nothings more important than a blueprint-- by setting a clear goal, it will save projects valuable time and money.
